# Chat GDI Virus

GDI Virus (non-malicious) made by YouTube chat!

## Files

- **main.py** — The main virus script with bytebeat
- **decode.py** — Encrypted command that was used for the YouTube chat
- **compress.py** — Converts the script into a base64-encoded payload
- **main.txt** — Messages for YouTube chat containing the command to launch the virus

## Setup

```bash
pip install -r requirements.txt
```
